# Getting Started

이 페이지는 로컬 환경에서 프로젝트를 빠르게 실행하는 방법을 안내합니다.

## 준비 사항

- Python 3.10+
- pip
- Git

## 빠른 시작

1. 저장소를 클론합니다.
2. 의존성을 설치합니다.

   ```bash
   pip install -r requirements.txt
   ```

3. 필요한 API 키나 환경 변수를 설정합니다.
4. 애플리케이션을 실행합니다.

   ```bash
   python main.py
   ```

5. `logs/` 아래에 로그가 생성되는지 확인합니다.
