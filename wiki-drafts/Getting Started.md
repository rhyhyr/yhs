# Getting Started

This page helps you run the project quickly on a local machine.

## Prerequisites

- Python 3.10+
- pip
- Git

## Quick Setup

1. Clone the repository.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Add required API keys or environment variables.
4. Run the app:

   ```bash
   python main.py
   ```

5. Confirm logs are generated under `logs/`.
