# 학기 프로젝트 제안서 

## 1. 프로젝트명

Global Campus Visa Navigator RAG: 외국인 유학생 비자·학사 안내 근거중심 도우미

## 2. 문제정의

외국인 유학생은 대학 생활 안내, 체류자격(비자), 출입국 신고 의무를 서로 다른 문서에서 확인해야 한다. 현재 데이터는 다음과 같이 성격이 다르다.

- 대학 안내형 문서: 2024 외국인유학생 대학생활안내(국문)
- 체류 자격 안내형 문서: 비자 내비게이터(국문)
- 법령형 문서: 출입국관리법 시행령(2025.06.01 시행)

문서마다 표현 방식이 달라 단순 검색으로는 질문에 맞는 조항과 절차를 빠르게 찾기 어렵고, 잘못된 이해는 행정 불이익으로 이어질 수 있다. 따라서 답변 자체보다 "근거 문서와 조항을 정확히 찾아 제시"하는 기능이 핵심이다.

## 3. 핵심 기능 (3개)

### 기능 1) 비자·학사 하이브리드 질의응답
- 벡터 검색(문맥 의미) + 그래프 검색(자격/전환/의무 관계) 결합
- Fast Path는 일반 안내, Deep Path는 법령/예외 케이스 처리
- 답변에 근거 문서명, 해당 절/조항, 인용 스니펫 포함

### 기능 2) 정책 문서 중심 인덱싱 파이프라인
- PDF 로딩, 정제, 청킹, 임베딩, 조항 단위 메타데이터 추출
- 문서 타입(대학안내/민원안내/법령)별 파싱 규칙 분리
- 개정일/시행일 기반 최신 버전 우선 노출

### 기능 3) 리스크 최소화 응답 정책
- 근거 부족 시 단정 답변 금지, "추가 확인 필요" 메시지 강제
- 민원 채널(하이코리아/1345 등) 안내를 자동 첨부
- 회귀 질문셋으로 정확도·지연시간·근거 충실도 자동 점검

## 4. 기술 스택

- Language: Python 3.11+
- Backend/Orchestration: Python CLI Runner
- LLM: Gemini/OpenAI/Ollama (환경별 선택)
- Retrieval: SentenceTransformers, Hybrid Retrieval (Vector + Graph)
- Graph DB: Neo4j
- Data Processing: pypdf/pdfplumber, custom cleaner/chunker/extractor
- Evaluation: 회귀 테스트, latency 로그, 근거 포함률 지표
- DevOps/Collab: Git, GitHub, Issues/Projects

## 5. 16주 마일스톤 (1학기 기준)

### 1-2주: 요구사항 구체화
- 외국인 유학생 질문 시나리오(비자/학사/생활) 60개 수집
- 금지 응답 정책(단정 금지, 근거 필수) 명세화
- 성공 지표 정의(정확도, 근거 포함률, SLA)

### 3-4주: 데이터 구조화 설계
- PDF 코퍼스 분류(대학안내/비자안내/법령)
- 조항/항/호 단위 메타데이터 스키마 설계
- 시행일/개정일 기반 최신성 우선순위 규칙 수립

### 5-6주: 인덱싱 파이프라인 구현
- 파서/청커/임베더/저장 파이프라인 구현
- 문서 해시 기반 증분 인덱싱 구축
- 초기 코퍼스 적재 및 품질 점검

### 7-8주: 하이브리드 검색 MVP
- 벡터+그래프 결합 검색 구현
- 근거 문서/조항 출력 포맷 구현
- 중간 데모(핵심 질문셋 기준)

### 9-10주: Fast/Deep 고도화
- 질문 유형별 라우팅 규칙 개선
- 법령 질문 시 Deep 우선 정책 적용
- 불확실성 메시지 자동 분기 강화

### 11-12주: 평가 자동화
- 회귀 질문셋 자동 실행
- SLA(Fast 5초, Deep 10초) 추적
- 저신뢰 케이스 수집 및 데이터 보강

### 13-14주: 운영 안정화
- 예외 처리 및 로그 체계 강화
- FAQ/민원 안내 템플릿 정교화
- 발표 시연 시나리오 완성

### 15주: 통합 점검
- 성능·정확도 튜닝
- 설치/실행/한계 문서화

### 16주: 최종 발표
- 데모 및 지표 결과 공유
- 확장 과제 제안(다국어, 웹 UI, 개인화)

## 6. 1학기-2학기 통합 확장안 (옵션)

- 1학기: 비자·학사 안내 코어 RAG + 평가체계 구축
- 2학기: 웹 서비스화, 다국어 응답, 사용자 피드백 기반 개선 루프
